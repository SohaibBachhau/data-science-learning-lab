# Probability

This directory contains permanent notes on random variables, probability distributions, moments, dependence and limit theorems used throughout statistics, econometrics and machine learning.

## Current foundation

Recommended reading order:

1. [Random variables and probability distributions](random-variables-and-distributions.md)
2. [Joint distributions, independence and iid sampling](joint-distributions-independence-and-iid.md)
3. [Expectation](expectation.md)
4. [Conditional expectation](conditional-expectation.md)
5. [Variance, covariance and moments](variance-covariance-and-moments.md)
6. [Law of large numbers](law-of-large-numbers.md)
7. [Central limit theorem](central-limit-theorem.md)

## Main learning chain

```text
random variables and distributions
→ joint and conditional distributions
→ expectation and conditional expectation
→ variance, covariance and moment conditions
→ law of large numbers
→ central limit theorem
```

## Connection to econometrics

Probability provides the language used to describe:

- the joint distribution of observations;
- conditional mean regression functions;
- exogeneity assumptions;
- sampling uncertainty;
- convergence of estimators;
- large-sample normal approximations.

Course-specific applications are stored under `courses/`, while the explanations in this directory are intended to remain useful across courses.
