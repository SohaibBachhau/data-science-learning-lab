# Advanced Econometrics Foundations

This learning programme develops the probability, statistics and
econometrics foundations needed for advanced econometrics, data science
and machine learning.

## Current topics

- Expectation
- Variance
- Conditional expectation
- Linear regression
- Exogeneity
- Correct specification
- Maximum likelihood
- Ordinary least squares
- Unbiasedness
- Consistency
- Law of large numbers
- Central limit theorem
- Asymptotic normality

## Permanent knowledge

The polished explanations and derivations developed during this
programme are stored in the relevant subject directories under
`knowledge/`.
